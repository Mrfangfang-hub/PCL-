## 1.投影.cpp

可以实现把三维点云投影到二维平面上，实现降维。通过把方程ax+by+cz+d=0中的a，b，c，d分别设置成a=b=d=0，c=1，从而得到z=0的平面，也就是每个点都失去z轴信息，从而完成投影。
